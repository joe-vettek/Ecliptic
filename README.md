# Ecliptic Seasons

![](https://media.forgecdn.net/avatars/thumbnails/1095/529/64/64/638640712387664433.png)

“节气”是Minecraft的一个中国风模组。

Ecliptic Seasons is a Chinese mod about seasons and weather.

### New

新增了API构建并重构了目录。

新增了群系缓存查询

新增了冬季白色地图颜色匹配

新增了缓冲查询，避免河流沙滩气候与周围群系不同。
